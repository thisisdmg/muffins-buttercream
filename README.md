# Muffins::Buttercream

Adds the Muffin Buttercream Theme to a Rails Application.

## Requirements

- Ruby >= 1.9.3
- Rails ~> 4.1

## Installation

Add this line to your application's Gemfile:

    gem 'muffins-buttercream', github: 'thisisdmg/muffins-buttercream'

execute:

    $ bundle install

and run:

    $ bin/rails g muffins:buttercream:install
